Selamat datang di website komunitas Roleplay.

Mainkan server RP terbaik dan nikmati pengalaman realistis.

Apa itu SAMP?

SAMP adalah multiplayer mod dari GTA San Andreas yang memungkinkan pemain bermain bersama secara online.

Apa itu Roleplay?

Roleplay adalah bermain dengan memerankan karakter seperti kehidupan nyata.

Server Rules

No Random Deathmatch (RDM)

Dilarang membunuh pemain tanpa alasan RP.

Good : Polisi menembak kriminal setelah kejar-kejaran.

Bad : Menembak pemain di jalan tanpa alasan.

No Vehicle Deathmatch (VDM)

Dilarang menabrak pemain dengan kendaraan.

Good : Tidak sengaja menabrak.

Bad : Sengaja menabrak pemain tanpa RP.

No Meta Gaming

Dilarang menggunakan info luar game.

Bad : Mengetahui nama dari nametag.

No Power Gaming

Dilarang melakukan hal tidak realistis.

Bad : Mengangkat mobil sendirian.
